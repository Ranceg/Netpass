Wireless Password Retrieval Through Connection History Tool
*Optimized for Windows operating system ONLY.
[Netpass.bat] Is an command terminal batch file scripted by Rance Garrett.
$ When run, this allows the user to regain the network passwords that their computer has previously connected to.
$ This also can show the user a detailed panel of the networks that the user has connected to.
$ The interface is simple and easy to navigate.
